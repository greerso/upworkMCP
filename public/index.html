Upwork MCP Server

Best-in-class remote MCP server for the Upwork GraphQL API.

MCP endpoint: /mcp (OAuth-protected)

See the README for setup, scopes, and how to connect Claude / other clients.

After deploying, set UPWORK_REDIRECT_BASE secret (or HOST) and update any registered callback URIs. Run `npm run validate` before deploys. See README "Production Readiness Notes".
